## Claude Projects 활용 가이드

1. **새 대화 시작 시** — 이 `README.md`와 `course_info.md`를 먼저 참고해 전체 맥락 파악
2. **강의자료 PDF** — `pdf/` 폴더의 파일 목록을 참고해 필요 시 Project Knowledge에 별도 업로드
3. **과제 마감** — `course_info.md`의 `⚠️` 표시 항목을 우선순위 높게 처리
4. **코드 참조** — 실습 코드는 `code/`, 과제 코드는 `labs/`, 보고서는 `assignment_report/`에서 검색
