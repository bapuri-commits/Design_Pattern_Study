# 프로젝트 지침

Claude 프로젝트 지침 사본을 여기에 붙여넣어 관리하세요.

## 워크플로우

1. 채팅에서 토론 → 노트 본문 생성(또는 직접 작성)
2. 옵시디언에 저장 → GitHub에 커밋·푸시
3. Claude 프로젝트에서 **Sync now**
4. 다음 대화부터 `project_knowledge_search`로 맥락 파악

- Claude는 **읽기 전용**. GitHub에 쓰는 것은 본인이 수행한다.
- 푸시 후 Claude 반영은 **수동 Sync**가 필요하다.

## 볼트 규칙 (요약)

- 새 노트는 우선 `inbox/`에 저장, 정리되면 `notes/`로 이동
- 관련 노트는 `[[링크]]`로 연결, 주제가 커지면 `MOC/`에 허브 노트 작성
- `status`, `author`는 frontmatter, 분류는 `#태그` 사용
